# Output Formats

Specification of all CSV output files generated by CLAW-Agent.

---

## Final Output Files

### all_shift_faults.csv

**Location**: `data/final_output/all_shift_faults.csv`

**Purpose**: Primary output - all verified faults

**Schema**:

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| FullTimestamp | datetime | Combined date + time | 2025-04-02 04:00:00 |
| timestamp | str | Time (HH:MM) | 04:00 |
| description | str | Fault description | RF issues |
| tag | str | Assigned tag | Accelerator |
| run_number | str | Run number (optional) | 4346807 |
| ShiftLogNumber | int | Logbook entry number | 4346807 |
| ShiftLogbookURL | str | URL to logbook | https://logbooks.jlab.org/entry/4346807 |
| ShiftTitle | str | Shift summary title | Shift Summary : 04.02.2025 Owl shift |
| ShiftDateTime | str | Summary creation time | 2025-04-02 01:23:00 |
| ShiftHall | str | Hall name | hall_c |
| FragmentLink | str | Clickable timestamp link | https://...#:~:text=04%3A00 |
| verification_status | str | "accurate" or "fixed" | accurate |

**Sample**:
```csv
FullTimestamp,timestamp,description,tag,run_number,ShiftLogNumber,ShiftLogbookURL,ShiftTitle,ShiftDateTime,ShiftHall,FragmentLink,verification_status
2025-04-02 04:00:00,04:00,RF issues,Accelerator,,4346807,https://logbooks.jlab.org/entry/4346807,Shift Summary...,2025-04-02 01:23:00,hall_c,https://...#:~:text=04%3A00,accurate
```

### manual_check.csv

**Location**: `data/final_output/manual_check.csv`

**Purpose**: Faults requiring manual review

**Schema**: Same as all_shift_faults.csv, minus `verification_status`

## Intermediate Files

### accurate.csv

**Location**: `data/verified/accurate.csv`

**Purpose**: Faults with verified timestamps

**Schema**: All columns from processed_summaries.csv

### inaccurate.csv

**Location**: `data/verified/inaccurate.csv`

**Purpose**: Faults needing timestamp correction

**Schema**: All columns from processed_summaries.csv

### fixed.csv

**Location**: `data/fixed/fixed.csv`

**Purpose**: Successfully fixed faults

**Schema**: Same as inaccurate.csv + `fix_confidence` column

### processed_summaries.csv

**Location**: `data/processed/processed_summaries.csv`

**Purpose**: Extracted and tagged faults

**Schema**:

| Column | Type | Description |
|--------|------|-------------|
| timestamp | str | Time (HH:MM) |
| description | str | Fault description |
| tag | str | Assigned tag |
| run_number | str | Run number |
| ShiftLogNumber | int | Logbook number |
| ShiftLogbookURL | str | Logbook URL |
| ShiftTitle | str | Summary title |
| ShiftDateTime | str | Summary datetime |
| ShiftHall | str | Hall name |
| FragmentLink | str | Timestamp link |

## Reading Output Files

### Python

```python
import pandas as pd

# Load final output
df = pd.read_csv("data/final_output/all_shift_faults.csv")

# Filter by tag
accelerator_faults = df[df['tag'] == 'Accelerator']

# Filter by date range
df['FullTimestamp'] = pd.to_datetime(df['FullTimestamp'])
recent = df[df['FullTimestamp'] >= '2025-04-01']

# Group by tag
tag_counts = df['tag'].value_counts()
```

### Command Line

```bash
# Count faults
wc -l data/final_output/all_shift_faults.csv

# View first 10 faults
head -11 data/final_output/all_shift_faults.csv

# Filter by tag (requires csvkit)
csvgrep -c tag -m "Accelerator" data/final_output/all_shift_faults.csv

# Extract descriptions
csvcut -c description data/final_output/all_shift_faults.csv
```

## Data Quality

### Validation

```python
def validate_output(df):
    required_cols = ['timestamp', 'description', 'tag', 'ShiftLogNumber']
    
    for col in required_cols:
        assert col in df.columns, f"Missing column: {col}"
    
    assert df['timestamp'].str.match(r'^\d{2}:\d{2}$').all(), "Invalid timestamp format"
    assert not df['description'].isnull().any(), "Empty descriptions"
    
    return True
```

### Common Issues

| Issue | Cause | Fix |
|-------|-------|-----|
| Empty descriptions | Extraction failed | Check LLM prompt |
| Invalid timestamps | Normalization failed | Check text_utils |
| Missing tags | Tagging failed | Check ChromaDB |
| Duplicate entries | Multiple runs | Deduplicate |
